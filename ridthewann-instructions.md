### Deployment

To deploy this release manually:

1. **Build the project** in Release mode:
   ```powershell
   cmake --build build --config Release
   ```

2. **Prepare the release folder** (`WeaRStudio_v0.1`):
   - Copy `build/bin/Release/WeaRStudio.exe`
   - Copy `build/bin/Release/*.dll` (Qt, FFmpeg, etc.)
   - Copy `plugins/` folder to `WeaRStudio_v0.1/plugins` (ensure ExamplePlugin.dll is inside)
   - Copy `LICENSE` and `README.md`

3. **Distribute** the folder (or ZIP it).
